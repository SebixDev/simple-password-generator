# Simple Password Generator

Ein webbasierter Passwort-Generator, der Benutzern hilft, sichere und zufällige Passwörter direkt im Browser zu erstellen.

## Projekt-Fortschritt und Abschluss

### Aufgabe 1: Grundfunktion (Abgeschlossen)
In dieser Phase wurden die Benutzeroberfläche und die Logik zur Generierung von Zufallspasswörtern implementiert.

### Aufgabe 2: Code-Refactoring und Struktur (Abgeschlossen)
Zur Übung der Web-Basics wurde das JavaScript aus der HTML-Datei extrahiert und in eine externe Datei (script.js) ausgelagert. Ziel war die praktische Anwendung von Clean-Code-Prinzipien durch die Trennung von Struktur und Logik.

## Zukünftige Überlegungen
* Es wäre eine Überlegung wert, das Skript irgendwann so zu verändern, dass auch andere Passwortlängen über die Benutzeroberfläche möglich sind.

## Verwendete Technologien
* HTML5 – Struktur der Anwendung
* CSS3 – Design und Layout
* JavaScript – Logik der Passwort-Erstellung (Modular)
* Git und GitHub – Versionsverwaltung und Hosting
